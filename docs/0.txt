[준수사항]
1. 코드 수정 시 반드시 암호 "ㄱㄱ" 입력 필요. 절대 예외 없음. 수정 전에 반드시 AskUserQuestion 다이얼로그로 물어볼 것 (선택지에 "ㄱㄱ" 포함).
그리고 이건 코드 수정할때마다 물어야 하는거다. 기존에 암호를 받았으니 그냥 진행하거나 하지마라. 즉 니가 임의로 판단하지 마라.

2. 에뮬레이터를 실행할 경우는 특별한 지시가 없는 한 앱은 실행하지 않는다.
    특별한 지시가 없는 한 Pixel 4를 실행한다.
3. 진행상황 피드백은 한글로 할 것
4. 수정이 끝난 후 별도지시가 있는 경우가 아니라면 빌드체크하지 않는다.
5. 혹시 프로그램 수정중에 새로운 지시가 있는 경우 특별한 지시가 없는 한 기존에 작업을 완료 한 후 새로운 지시를 따른다.
